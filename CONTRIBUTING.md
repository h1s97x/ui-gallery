# Contributing to UI Gallery

Thank you for your interest in contributing!

## How to Contribute

### Adding a New Design

1. Create a new folder under the root directory with a lowercase, hyphenated name (e.g. `my-design/`).
2. Include `README.md` in your design folder describing the design system.
3. Ensure all HTML files are self-contained or reference files via relative paths.
4. Submit a pull request with a clear description.

### Folder Naming Convention

- Lowercase, hyphen-separated: `my-design/`
- No spaces, no `Copy` suffix
- Use the design system's common name in English

### Commit Messages

Follow [Conventional Commits](https://www.conventionalcommits.org/):

```
feat: add acme design system
fix: correct volcengine preview path
chore: update README links
```
